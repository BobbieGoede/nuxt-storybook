## Contributing

1. Install dependencies with `pnpm`.
2. Run `pnpm dev:prepare` to generate stubbed `dist` directory.
3. Make your changes.
4. Run `pnpm lint` and `yarn test` to verify that there is no issues (consider adding new test for your changes).
5. Submit a PR.
